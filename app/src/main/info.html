OSMZ II

Information

Here is the summary of the assignment - OSMZ II:

  1. The task involves extending the functionality of an HTTP server for an Android application.
  2. The server serves documents located either in the internal storage or on the SD card of the device/emulator.
  3. The task requires creating a simple static website with HTML files and images to test the transfer of binary data.

The HTTP server is to be modified to parse client/browser requests, specifically focusing on the GET method and the requested document path. If the requested file exists, the server returns a 200 OK status code along with the file content. If the file does not exist, it returns a 404 Not Found error along with a simple error page. The server response headers must include correct Content-Type and Content-Length values.
